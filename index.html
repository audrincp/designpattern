PROPRIEDADE object-fit #Ir para o topo

  • Sintaxe: fill | contain | cover | none | scale-down
  • Valor Inicial: fill

A propriedade object-fit específica como o conteúdo de um elemento substituto deve ser ajustado à caixa por sua altura e largura utilizadas

Você pode querer ou precisar que todas as imagens (ou vídeos, ou inputs, etc.) em uma página tenham os mesmos valores de altura e largura. Um exemplo disso é imagens de avatar em perfis de usuários ou depoimentos

As imagens que você acaba usando na página podem não ter as mesmas dimensões, nem a mesma proporção. Isso é comum em aplicativos nos quais o usuário pode carregar e usar suas próprias imagens que precisam se ajustar a pontos específicos com dimensões predefinidas.

Usando a propriedade object-fit, você pode ajustar o conteúdo de uma imagem nas dimensões especificadas em sua folha de estilos.

O conteúdo pode ser configurado para os valores fill, contain, cover, none ou scale-down.

VALORES

fill

O conteúdo do elemento substituído é dimensionado (dimensionado para cima ou para baixo, esticado ou reduzido) para preencher a caixa de conteúdo do elemento definida usando as propriedades width e height. A proporção do conteúdo não é preservada enquanto é dimensionada para preencher o elemento.

                                                   img {
                                                       object-fit: fill;
                                                   }

contain

A proporção do conteúdo do elemento substituído é preservada e aumentada o máximo possível, permanecendo dentro dos limites do elemento, definido por sua altura e largura.

                                                    img {
                                                        object-fit: contain;
                                                    }

Imagem Original

Imagem após o valor contain atribuído:

cover

A proporção do conteúdo do elemento substituído é preservada e aumentada o suficiente para preencher toda a caixa de conteúdo do elemento definida pela largura e pela altura.

                                                    img {
                                                        object-fit: cover;
                                                    }

Imagem Original

Imagem após o valor cover atribuído:

none

O conteúdo substituído não é redimensionado para caber dentro da caixa de conteúdo do elemento.

                                                     img {
                                                         object-fit: none;
                                                     }

Imagem Original

Imagem após o valor none atribuído:

scale-down

Dimensiona o conteúdo como se none ou contain fosse especificado, o que resultaria em um tamanho menor de objeto concreto.

De acordo com a especificação: o tamanho do objeto concreto é o resultado da combinação das dimensões intrínsecas e do tamanho especificado de um objeto com o tamanho de objeto padrão do contexto no qual ele é usado, produzindo um retângulo com largura e altura definidas.

                                                      img {
                                                          object-fit: scale-down;
                                                      }

PROPRIEDADE transform#Ir para o topo

Transform CSS permitem que elementos estilizados com CSS sejam transformados em espaço bidimensional. Esta especificação é a convergência das transformações CSS 2D e das especificações de transformações SVG .
